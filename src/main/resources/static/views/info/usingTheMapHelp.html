Using the Map - help

A map perspective is provided for each entity when geospatial nature is available. The map can be used for displaying entities on it (pins), selecting instances or setting geographical bounding box to be used in the query. Further features are zoom in & out and searching for toponyms.

In general pins can be shown by holding ctrl and dragging on the map. The displayed red pins are not, selected just shown. Each pin can be selected by clicking on it (becomes green) and represents a related entity (under which the globe icon button was clicked while opening the map).

In contrast with selecting pins individually, it is feasible to set a geographical bounding box instead. Bounding box can be drawn by clicking on the "" button and then clicking in two points on the map consequently. The drawn rectangle will stay there and the respective bounding box will be used in the query for this related entity. By default when drawing a bounding box, the entities (pins) within it appear on the map. These pins are not selected, they are just there to ensure that the bounding box set includes instances. If clicking on any pin, it will turn into green by making it selected and the bounding box set will be removed. In each case you cannot have both instances selected and bounding box set at the same time. In any case a drawn bounding box can be cleared by pressing the "" button.
